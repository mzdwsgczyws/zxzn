# 道性小程序 · UI 设计系统（紫金墨 · 玄学高级感）

本文与仓库内 **`.cursor/skills/dao-ui-design/SKILL.md`** 配套，供新增页面、改版、Agent 实现功能时对照。

---

## 1. 品牌气质

| 关键词 | 设计转译 |
|--------|----------|
| 玄学 / 道教意象 | 深色靛紫为「玄」，金为「阳、贵、界」，纸本米色为「经卷、静」；避免廉价霓虹与高饱和渐变堆叠。 |
| 高级感 | 大留白、细金边、弱阴影、少装饰；标题字重清晰、正文灰褐易读。 |
| 可信与克制 | 免责声明用小字、低对比灰；主 CTA 明确但不吼叫。 |

**禁忌**：大面积纯黑底 + 高亮荧光字；过多 emoji；与「医疗/命理断言」相关的强红警示色（除非真错误态）。

## 3. 字体与层级

| 角色 | rpx 参考 | 字重 | 色 |
|------|----------|------|-----|
| 页面大标题 | 40 | 600 | `--dao-purple-deep` |
| 导航/模块标题 | 34–36 | 600 | 顶栏内 `--dao-gold-pale`；内容区 `--dao-purple-deep` 或 mid |
| 卡片标题 | 28–32 | 600 | `--dao-purple-mid` |
| 正文 | 26–28 | 400 | `--dao-ink` |
| 辅助说明 | 22–24 | 400 | `--dao-ink-muted` / `--dao-ink-faint` |
| 脚注/免责 | 20–22 | 400 | `--dao-ink-faint` |

**行高**：正文 `1.55–1.65`；标题下说明 `1.45–1.55`。

**字间距**：仪式感短标题可略加 `letter-spacing`（如首页 `6rpx`）；长段落勿过大。

## 4. 组件规范

### 4.1 卡片 `.card`（全局）

- 圆角 `20rpx`，内边距 `32rpx`，外边距 `24rpx`。
- 金边 + 淡紫阴影，与 `app.wxss` 保持一致。

### 4.2 按钮

| 类名 | 用途 |
|------|------|
| `.btn-primary` | 主操作：靛紫渐变底、白字、大圆角。 |
| `.btn-ghost` | 次要：透明底、靛紫描边与字。 |

**规范**：同一屏主 CTA 仅一个 primary；破坏性操作若需按钮，另议（当前项目少用）。

**禁用态**（若实现）：降低透明度 `0.45` 或灰化描边，勿用高饱和红作主按钮。

### 4.3 表单

- `label`：26rpx，`--dao-ink-muted`。
- `slider`：保持微信原生；重要字段上下留白与其它 field 一致。

### 4.4 分割线

- 浅褐 `#ece5d8` / `#e8e0d4`，1rpx；区隔模块而非切碎版面。

## 5. 页面类型与导航

- **系统导航栏**（`app.json`）：`navigationBarBackgroundColor` 与 `--dao-purple-deep` 一致，白字。
- **首页自定义顶栏**：深紫 ritual 渐变 + 底边金线，与系统栏二选一出现时保持「深紫 + 金」同一套叙事，避免第三种无关蓝紫。

---

## 6. 插图与 Canvas

- 心象配图等第色见 `lot-art.js`（上上金、上红、中蓝…），与 UI 令牌中的「金/紫」叙事兼容即可，不必强行改成靛紫。
- 自修五行雷达：数据面用半透明靛紫，网格墨褐灰，与卡片体系一致。

---

## 7. 可访问与合规文案

- 关键免责：**小字号 + 低对比**，与主内容区分。
- 避免用颜色作为唯一信息区分（分数配合数字与文案）。

---

## 8. 当前代码中的统一优化建议（清单）

以下可在改到对应文件时顺手替换为 CSS 变量或合并为全局类：

1. **重复 hex**：`#1a237e`、`#3949ab` 在 `track`、`profile`、`result`、`theory-intro`、`lottery` 等 wxss 中多次出现 → 改用 `var(--dao-purple-deep)` 等。
2. **渐变按钮**：`personality/result`、quiz 进度条等局部 `linear-gradient` → 优先复用 `--dao-purple-mid` / `--dao-purple-soft` 组合，或与 `.btn-primary` 同向 135° 渐变。
3. **卡片**：若某页自建白底卡片，对齐 `.card` 的圆角、金边、阴影三件套，避免另一种「纯白无框」。
4. **muted 文案**：统一用 `--dao-ink-faint` 或 `--dao-ink-muted`，减少 `#8d7b68` / `#9a8b7a` / `#78909c` 混用（可映射到 faint/muted）。
