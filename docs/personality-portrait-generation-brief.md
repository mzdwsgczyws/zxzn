# 道性十六型 · 意象人物图（交给豆包等工具生成用）

> 程序内「历史形象」仅为**文化取意**，与真实人物外貌、肖像权无关。生图时请使用**原创虚构角色**，体现**气质意象**，避免生成可辨认的真人照片或侵犯第三方 IP 的卡通形象。

---

## 一、微信小程序侧技术建议（请按此导出资源）

| 项目 | 建议 |
|------|------|
| **画幅比例** | **1:1 正方形**（与结果页 280×280 展示、分享卡片右侧图块、海报 300×300 区域一致） |
| **推荐像素** | **1024×1024** 或 **768×768**（上线前用工具压到展示够用即可，如长边 **512～768**） |
| **格式** | **PNG**（带柔和背景）或 **JPG**（质量 85% 左右，体积更小） |
| **单张体积** | 压缩后尽量 **≤ 200～400 KB**，减轻主包/分包体积 |
| **安全区** | 主体（头肩）放在**中上区域**，**底部约 10%** 少放关键细节（避免被圆角卡片裁切） |
| **背景** | 可用淡墨晕染或低对比场景，**避免纯白底+细线**（小图缩略易糊） |
| **命名** | 保存为 **`0.png`～`15.png`**（与下表 `id` 一致），放入小程序 `images/personality-portraits/` |

## 二、统一画风与内容要求（每条生图都加上）

**风格关键词（可复制到每条提示词末尾）：**

> 中国画水墨写意为主，留白与晕染；少量赛博元素（如衣领/背景中极细的青蓝、品红微光线路或几何点缀），**不要**全屏科幻 HUD；**半身或胸像**，**原创虚构人物**，**不要**写实名人脸、不要文字、不要水印、不要二维码。

**宜避免：**

- 与某历史人物照片高度相似的可识别面容  
- 血腥、迷信符咒、宗教冒犯性符号  
- 过于拥挤的 UI 风边框、大面积高饱和霓虹
